## Iniciando o projeto:

- Clonar o projeto do github em um diretório de sua preferência
- Abra o Terminal na pasta do projeto e digite: npm start

```
$ git clone https://github.com/rafaelspirlandelli/api-filmes
$ cd api-filmes
$ npm start

```
